# Book Recommendation System (C++ | OOP | Academic Project)

## Project Overview
This is a simple **Book Recommendation System** developed using **C++** and **Object-Oriented Programming (OOP)** concepts.  
The system recommends books to users based on data stored in a file-based database.

This project is created for academic and learning purposes.

---

## Team Name
**The Semi-Colons**

---

## Features
- Store book details (title, author, genre, rating)
- Store user preferences
- Recommend books based on user interest
- File-based DBMS using text files
- Proper use of OOP principles

---

## Technologies Used
- Language: C++
- Concepts:
  - Classes and Objects
  - Encapsulation
  - Inheritance
  - Polymorphism
  - File Handling

---

## Recommendation Logic
Books are recommended based on:
- User-selected genre
- Book ratings
- Preferences stored in files
